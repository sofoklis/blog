title: Functors date: 2013/04/07 20:45:00 -0800 tags: [{"name":"functors","id":125268712},{"name":"category theory","id":125268711}] author: Sofoklis Papasofokli alias: /functors ## Functors A functor requires the following 2 properties: 1. A contructor or apply method which for any type A, we can construct a type in the new category T[A] 2. map method which given a transformed type T[A] and a function that transforms A=>B we can construct T[B]
